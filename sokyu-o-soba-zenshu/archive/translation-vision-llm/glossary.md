# Glossary of Recurring Terms

This glossary provides consistent English translations for Japanese trading
terminology used throughout the 宗久翁相場全集.

## Market structure

| Japanese | Romanization | English | Notes |
|----------|-------------|---------|-------|
| 相場 | sōba | market / market price | General term for commodity market or price |
| 米相場 | kome sōba | rice market | |
| 天井 | tenjō | ceiling / market top | The highest point of a price move |
| 底 | soko | bottom / market floor | The lowest point of a price move |
| 大底 | ōzoko | major bottom | |
| 半値 | hanne | half-price / midpoint | The price midway between top and bottom |
| 値段 | nedan | price / price level | |
| 高値 | takane | high price | |
| 安値 | yasune | low price | |
| 極相 | gokusō | extreme / price extreme | The point where a move exhausts itself |
| 大勢 | taisei | overall trend / prevailing direction | The dominant direction of the market |
| 持合ひ | mochiai | consolidation / standstill | Period of sideways movement |
| 上り相場 | nobori sōba | rising market / bull market | |
| 下り相場 | kudari sōba | falling market / bear market | |
| 落値 | ochine | falling price | |

|----------|-------------|---------|-------|
| 賣買 | baibai | buying and selling / trading | |

| 賣り | uri | selling / a short position | |
| 仕掛く | shikakeru | to enter a position / to initiate a trade | |
| 手仕舞ふ | teshimau | to close a position / to exit | |
